# Hot Ice

Hot Ice is a mobile-first daily sports player guessing game.

## Public Beta v41 Position + Daily Save + Share

Updates:
- NFL exact position now ranks above division
- MLB exact position now ranks above division
- Same team remains the strongest signal
- Daily puzzle state is saved in localStorage
- Closing/reopening the browser restores guesses, timer, clues, solved/revealed status, and streak result
- Once the daily puzzle is solved or revealed, the input locks until the next daily puzzle
- Share Result opens the phone share sheet when available, including Messages on iPhone
- Copy Result copies the result text for texting/pasting
- Share links now use https://playhotice.com/
- Keeps expanded player pool, NFL/MLB refresh, mobile layout, streaks, transparent logo, and analytics
